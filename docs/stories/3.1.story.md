# Story 3.1: Public Properties Listing Page

**Epic:** 3. Public Listings
**Status:** Done Since 2026-01-14

## Story Statement
As a potential buyer (unauthenticated user), I want to view a beautiful gallery of all available properties on a dedicated page so that I can browse the market without logging in.

## Acceptance Criteria
1. Public API Endpoint: GET /api/v1/properties/published (No Auth).
2. Public Properties Page: Route / (Home).
3. Property Card Design: Thumbnail, Price, Title.
4. Navigation: Browse Properties link.

## Tasks
1. Backend: properties.py -> get_published.
2. Frontend: PropertyCard.tsx, PublicPropertiesPage.tsx.
