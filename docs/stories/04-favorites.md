# Story 04: Favorites Functionality

**Story ID:** FAV-001  
**Epic:** User Experience  
**Priority:** P1 - High Value  
**Estimate:** 5 points  

---

## User Story

As a **property seeker**, I want to **save properties without logging in**, so that **I can track properties across sessions**.

---

## Acceptance Criteria

- [ ] Heart icon on property cards
- [ ] Click heart to add/remove favorites
- [ ] Works without login or reload
- [ ] Toast notification on add/remove
- [ ] Favorites page at `/favorites`
- [ ] Shows all saved properties
- [ ] Navigation badge with count
- [ ] Persists in localStorage
- [ ] Survives refresh and reopen
- [ ] Empty state when no favorites
- [ ] Responsive on mobile/desktop
- [ ] Heart icon shows correct state

---

## Definition of Done
- [ ] localStorage integration tested
- [ ] Favorites page displays correctly
- [ ] Performance verified
- [ ] Mobile responsive
